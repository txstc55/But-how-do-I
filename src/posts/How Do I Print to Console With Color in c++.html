If you want to add color to your output in console, you will need to use ncurses. In c++, it's as easy as an include:

#include <ncurses.h>

Now you will need to define a color and initialize it as an attribute. You can do it using the following code:

// the first argument is an attribute number
// the next three are rgb values
init_color(21, 0, 100, 50);

// the first input is the attribute number
// the next two are:
// the forground color, which we initialized with attribute 21
// and the background color, here we use -1 to denote that transparent
init_pair(21, 21, -1);

Now that we initialized the color, the last thing is to actually use it before printing.

To achieve this, we will need to turn on the attribute, before the printing:

// turn on the attribute 21
// which is the color we defined
attron(21);

// now add the character to console output
addch(matrix.char_map[i][j]);

// if you want to remove the color
// just turn of this attribute
// remember there can be multiple attributes at the same time
// right now we are only demonstrating the color attribute
attroff(21)
